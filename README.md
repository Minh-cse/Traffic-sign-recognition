# Traffic - sign - detection 

## Introduction
This project is a traffic sign recognition system in real time.
The project aims towards assisting drivers in recognizing traffic signs  

## Features
* Realtime recognition
* Instant feedback
* Intuitive UI

## Tech Stack
### Frontend
* **React**
* **Tailwind CSS**
* **Axios**

### Backend
* **Google Colab**
* **Flask**
* **Pyngrok**
* **YOLOv8**

### Step 2: Start Frontend (on local)
1. Clone the repository:
```bash
git clone https://github.com/Minh-cse/Traffic-sign-recognition.git
cd Traffic-sign-recognition
```
3. Install npm
```bash
npm install
```
5. Run frontend
```bash
npm run dev
```
and the link should be http://localhost:5173/
